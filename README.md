₊⊹  INICIANDO O PROGRAMA: ₊⊹  
⏔⏔⏔⏔⏔⏔ ꒰ ᧔ෆ᧓ ꒱ ⏔⏔⏔⏔⏔⏔  

1. Abra o terminal na pasta do projeto (onde estão fitplanner.py e funcoes.py).

2. Execute: python3 fitplanner.py — o programa cria automaticamente as pastas e arquivos necessários.

3. O menu principal aparece. Digite o número da opção e pressione Enter para navegar.

4. Em qualquer submenu, opção 0 sempre volta ao nível anterior ou fecha o programa.

₊⊹  1 -    TREINOS        ₊⊹  
⏔⏔⏔⏔⏔⏔ ꒰ ᧔ෆ᧓ ꒱ ⏔⏔⏔⏔⏔⏔  

1. Novo treino  
Cria um treino preenchendo: nome, tipo/categoria, duração (em minutos), objetivo e observações. Em seguida, pergunta se deseja adicionar exercícios ao treino — você escolhe a categoria e seleciona da lista existente.

3. Ver treinos  
Lista todos os treinos salvos numerados. Digite o número de um treino para ver os detalhes completos: nome, tipo, duração, objetivo e observações.

4. Editar treino  
Selecione o treino e depois o campo que deseja alterar (nome, tipo, duração, objetivo ou observações). O novo valor substitui o anterior.

4. Excluir treino  
Exibe os detalhes do treino escolhido e pede confirmação antes de apagar. Operação irreversível.
