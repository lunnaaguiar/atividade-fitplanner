  ₊⊹  INICIANDO O PROGRAMA: ₊⊹  
       ⏔⏔⏔ ꒰ ᧔ෆ᧓ ꒱ ⏔⏔⏔  

1. Abra o terminal na pasta do projeto (onde estão fitplanner.py e funcoes.py).

2. Execute: python3 fitplanner.py — o programa cria automaticamente as pastas e arquivos necessários.

3. O menu principal aparece. Digite o número da opção e pressione Enter para navegar.

4. Em qualquer submenu, opção 0 sempre volta ao nível anterior ou fecha o programa.
